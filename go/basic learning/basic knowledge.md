# go开发环境搭建

## go环境变量与工作目录

bin：存放编译后生成的可执行文件

pkg：存放编译后生成的包文件

src：存放项目源码

## go常用命令

go run:运行程序

go build:测试编译

go fmt:格式化源码

go install:编译包文件并编译整个程序

go test:运行测试文件

go doc:查看文档

### go语言的"Hello world!"

```go
   package main
   import(
       "fmt"
   )
   func main(){
      fmt.println("Hello world!你好，世界！")
   }
```
